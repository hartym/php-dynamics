Glossary
========

.. glossary::

   package
      A dynamics package is coherent sets of assets.

   packageset
      A packageset is a coherent set of packages, providing a full set of
      complementary functionalities. Packagesets contains javascript code,
      associated stylesheets, images, and configuration needed to use the
      whole.

      For example, if you install the «jQuery» packageset, you'll be able
      to <import resource="path/to/jquery.xml" /> that will make available
      the «jquery» package.
