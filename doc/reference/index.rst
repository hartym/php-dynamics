PHP Dynamics
============

Welcome to PHP Dynamics documentation.

PHP Dynamics is a PHP 5.2 tool to manage asset management in PHP projects.

Features
========

It supports extensible configuration, dependency management, content filtering
(packing, path resolvers, ...), CDN usage, caching, and much more.

Library is not tied to any JavaScript framework, you can use about any of them.
