# Bundled configuration files

A set of packagesets are bundled with the plugin. Here is the list, and status of each one.

## dynamics.xml

Empty default configuration file. If you install (and enable) the plugin but do
not provide another dynamics.xml file within your project or the current app,
it will be the main one, and so the plugin will do nothing.
